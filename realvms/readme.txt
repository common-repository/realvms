== Description ==

RealVMS is a powerful video blogging plugin for WordPress that enables you to import videos, descriptions, tags, comments, and much more from YouTube and to place them directly into your own website in order to greatly enhance your own user experience.

You can quickly and easily use RealVMS to build automated blog networks or to enhance your existing websites with the power of online video. RealVMS incorporates an integrated social bookmarking feature unseen in other plugins, this feature will allow you to quickly and easily bookmark your blog posts on Del.icio.us and Faves. 

== Installation ==

1) Deactivate RealVMS from the 'Plugins' menu in WordPress if you have already activated it<br><br>
2) Upload the RealVMS folder to the /wp-content/plugins/ directory and overwrite any old files you may have there.<br><br>
3) Activate the plugin through the Plugins menu in WordPress
